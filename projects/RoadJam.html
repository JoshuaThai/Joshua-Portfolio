Road Jam is a game that I developed with a friend for the Global Game Jam's Next Gen Expert Jam. This game jam was a three-month long game development competition that started in November 2025 and ended in March 2026. For this game jam, we had to create a game that addresses one of three themes: road safety, coral reef preservation, and hunger. We decided to create a game that addresses road safety. In this game, players have to dodge traffic and obstacles on a highway while trying to get to their destination. Players need to avoid speeding or risk getting arrested. The purpose of the game is to raise awareness about the dangers of speeding and reckless driving, which is an epidemic in my local community. This game was developed using Godot and its built-in scripting language, GDScript.

This game was submitted to the Global Game Jam's Next Gen Expert Jam and was a finalist in the competition for the "Own the Road" category. I am very proud of this project and I am excited to see where it will lead me.
