Patient_Zero.exe is a game that I developed with a friend for the Gamedev.js 2026 game jam. This jam is an annual online hackathon focused on building HTML5 web games using JavaScript or web-friendly technologies within a 13-day window. We created a story-driven horror game where players take on the role of a character trapped in a virtual world generated by a rogue AI. Players must navigate through eerie environments, solve puzzles, and uncover the dark secrets of the AI's creation to escape. This game is meant to be a psychological horror experience that explores themes of isolation, identity, and the consequences of unchecked technological advancement. This game was developed using Godot and its built-in scripting language, GDScript.

This game is intended to be the debut game title for a game studio that I created with a friend. It is called Dead Signal Studios. We are planning to create more story-drive and horor games in the future and we are very excited to see where this studio will lead us.
